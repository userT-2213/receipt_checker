# 授業問題復習アプリ (Lecture Review Quiz App)

教員が登録した授業概要やアップロードした資料を基に、学生が復習問題を生成して回答し、自動採点や解説を通じて効率的に学習できるアプリケーションです。

---

## 💡 概要
本アプリは、日々の参考書や授業概要をインプットとして、AIが自動で復習問題を生成する学習支援ツールです。学生は生成された問題に回答し、自動採点と詳細な解説を確認することで、効率的な知識の定着を図ることができます。

※本プロジェクトでは、セキュリティの担保や高速なレスポンス対応は非目標（スコープ外）としています。

## ✨ 機能要件

### サブ機能一覧
1.  **授業概要・資料アップロード機能**
    * 教員（またはユーザー）が授業の概要テキストや、参考資料（PDFやテキスト等）をシステムにアップロードする機能。
2.  **AIによる復習問題自動生成・編集機能**
    * アップロードされたデータを基に、AIが自動で復習問題（選択式や記述式など）を生成する機能。
    * 生成された問題を必要に応じて編集・調整する機能。
3.  **問題回答・自動採点・解説確認機能**
    * 学生が生成された問題にシステム上で回答する機能。
    * 回答後、即座に自動採点が行われ、AIによる解説を確認できる機能。
4.  **学習履歴・採点結果閲覧機能**
    * 過去に解いた問題の採点結果や、学習の進捗履歴を一覧・詳細で確認できる機能。
5.  **問題や解答の修正機能**
    * 間違っていた問題データや、不適切な解答・解説を後から修正・ブラッシュアップできる機能。

### 🚫 スコープ外（作らないもの）
* リアルタイムでの学生同士のクイズバトルといった対戦機能
* 高度なセキュリティ対策（認証の厳格化、データ暗号化など）
* 高速なレスポンス対応（パフォーマンスの最適化やチューニング）
